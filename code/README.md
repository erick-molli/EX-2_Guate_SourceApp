# SA_RProject

The `code` folder has three different codes:
1.`SA_data_cleaning` is a Quarto file that contains all the steps for data manipulation. This step is initial and produces the `concentration_final` and `uncertainty_final` files that are inputs for the PMF software.
2.`SA_data_analysis` is a Quarto file used for additional analysis of the outputs from the PMF software. This produces graphs and statistical comparisons.
3.`stove use analysis` is an R file used to answer questions from the reviewers of the manuscript.
